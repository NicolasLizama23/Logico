# Manual de uso - Prototipo funcional LogiCo

## Alcance

Este prototipo corresponde a la Iteración/Sprint 0 del proyecto LogiCo. Implementa los mantenedores base del módulo de administración:

- Gestionar farmacias
- Gestionar motoristas
- Gestionar motos

No se incluyen todavía asignaciones, movimientos, reportes ni seguridad/inicio de sesión, ya que esos módulos pertenecen a iteraciones/sprints posteriores.

## Uso general

1. Ingresar a `http://localhost/logico-prototipo/`.
2. Seleccionar el mantenedor desde el menú superior.
3. Usar el botón `Agregar` para registrar un nuevo dato.
4. Completar los campos obligatorios.
5. Presionar `Guardar`.
6. Verificar que el registro aparezca en el listado.
7. Usar el buscador para filtrar registros.
8. Usar `Modificar` para editar un dato existente.
9. Usar `Eliminar` para borrar un registro.

## Evidencias sugeridas para el informe

- Pantalla de inicio del módulo de administración.
- Listado de farmacias.
- Formulario para agregar farmacia.
- Mensaje de farmacia agregada correctamente.
- Búsqueda de farmacia.
- Modificación de farmacia.
- Eliminación de farmacia.
- Repetir el mismo flujo con motoristas y motos.
- Captura de validación cuando se intenta guardar un formulario incompleto.
